# Lesson 4 - PHP and HTML

## Maqsad

Bu darsda siz:

- HTML ichida PHP yozishni
- PHP ichida HTML chiqarishni
- GET so'rovlarini ishlatishni o'rganasiz

---

## Misol

URL:

http://localhost:8505/?givenName=John&familyName=Doe

Natija:

Your given name: John  
Your family name: Doe

---

## Vazifa 1

Form orqali foydalanuvchining ism va familiyasini oling.

Agar foydalanuvchi ma'lumot kiritsa:

Hello John Doe

chiqsin.

Agar kiritmasa:

Please fill the form

chiqsin.

---

## Vazifa 2

HTML ichida PHP yozing.

Shunday yozing:

Agar name mavjud bo'lsa:

Welcome John

chiqsin.

Aks holda:

Guest

chiqsin.

---

## Vazifa 3

skills[] orqali ma'lumot yuboring:

?skills[]=PHP&skills[]=HTML

HTML ichida ro'yxat chiqaring:

- PHP
- HTML

Agar bo'sh bo'lsa:

No skills
